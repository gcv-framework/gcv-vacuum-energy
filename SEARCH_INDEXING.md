# Search and Indexing Notes

This repository is intended to be a public navigation layer for GCV.

## Recommended linking pattern

- Homepage route-map page → GitHub route-map repository
- GitHub route-map repository → Zenodo DOI records and OSF mirrors
- YouTube descriptions → homepage route map + GitHub route map
- OSF wiki → GitHub route map + canonical DOI records
- Zenodo descriptions, where possible → GitHub route map and homepage route page

## Google / normal web search

A clean GitHub README and a matching homepage route-map page can help normal web discovery. A sitemap and Search Console submission for the homepage route-map page are recommended.

## Google Scholar

Google Scholar indexing is less predictable for repositories and mixed archival records. The best chance for Scholar indexing comes from article-like landing pages with clear metadata, PDF links, titles, authors, dates, and citation information.
