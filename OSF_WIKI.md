# GCV Framework OSF Mirror

This OSF project provides backup and provenance mirrors for the Gauged Constant Vacuum-Mode (GCV) framework records.

GCV is a flux-fixed GR framework in which the strictly spacetime-constant vacuum mode is assigned to a global/flux sector, while local excitations remain in the local stress-energy tensor. The programme is organized into modular records covering radiative stability, residual Λ_eff, UV benchmark anchoring, observed-value localization, positive-Λ realization, matter-branch/shell mechanisms, and survey-facing falsifiers.
